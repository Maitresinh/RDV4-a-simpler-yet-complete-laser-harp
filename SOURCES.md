# Sources étudiées

## M1r3o/Laser-Harp-Project

https://github.com/M1r3o/Laser-Harp-Project

Source principale pour le principe galvo : positions discrètes, blanking, détection synchronisée, convertisseur DAC et logique de notes. Certaines versions sont sous CC BY-NC-SA. Aucun code source n'est copié dans ce dépôt.

## Trasselfrisyr/laserharp

https://github.com/Trasselfrisyr/laserharp

Source pour les pratiques Teensy, USB-MIDI, états de notes, gammes et hystérésis. Aucun fichier source n'est repris.

## Evanthx/laserharp

https://github.com/Evanthx/laserharp

Référence pour le découpage de logique testable et les tests unitaires.

## cybercraftics/laser_harp

https://github.com/cybercraftics/laser_harp

Référence MIT pour l'organisation mécanique, les pièces imprimables et l'idée d'octaves par mesure de distance. La tête de balayage est conçue pour un moteur pas-à-pas et n'est pas reprise pour les galvos.

## ThingEngineer/Laser-XY-Scanner

https://github.com/ThingEngineer/Laser-XY-Scanner

Référence pour un support existant de module AixiZ 12 mm. Les STL et CAD sont publiés sur https://www.thingiverse.com/thing:2710048 sous licence CC BY-SA 4.0 annoncée par le dépôt. RDV4 ne redistribue pas ces fichiers ; il les référence comme option mécanique existante.

## Pololu VL53L4CD Arduino

https://github.com/pololu/vl53l4cd-arduino

Bibliothèque utilisée pour le capteur d'octave en lecture continue non bloquante.
