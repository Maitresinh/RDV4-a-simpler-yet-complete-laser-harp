# Checklist avant commande

## Galvo

- [ ] boucle fermée ;
- [ ] support métallique inclus ;
- [ ] deux drivers inclus ;
- [ ] alimentation incluse ;
- [ ] photo lisible des entrées ;
- [ ] tension d'entrée indiquée ;
- [ ] entrée simple ou différentielle identifiée ;
- [ ] miroir visible compatible 650 nm.

## Laser

- [ ] puissance annoncée 1 mW ou moins ;
- [ ] puissance réellement vérifiable ;
- [ ] entrée TTL compatible 3,3 V ;
- [ ] alimentation 5 V ;
- [ ] diamètre du corps connu, idéalement 12 mm ;
- [ ] lentille réglable ;
- [ ] support métallique disponible ou modèle 3D existant compatible.

## Modules

- [ ] MCP4822 monté sur breakout ;
- [ ] OPT101 véritable ou vendeur traçable ;
- [ ] tension de sortie OPT101 compatible avec ADC 3,3 V ;
- [ ] VL53L4CD et non VL53L0X substitué discrètement ;
- [ ] Teensy officiel PJRC.

## Mécanique

- [ ] planche de contreplaqué ;
- [ ] entretoises nylon assorties ;
- [ ] équerres métalliques ;
- [ ] tube noir pour OPT101 ;
- [ ] capot ou boîte pour alimentations et drivers ;
- [ ] arrêt d'urgence montable en façade.

Ne pas commander le module d'interface galvo avant d'avoir confirmé le type d'entrée du kit choisi.
